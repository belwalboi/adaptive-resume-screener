# Presentation Member Guides

These files are detailed preparation notes for each team member before the teacher presentation and viva.

Files included:
- `SHUBH_GUIDE.md`
- `ADITYA_GUIDE.md`
- `MRINALENDU_GUIDE.md`
- `HARSHIT_GUIDE.md`
- `RITIK_GUIDE.md`

Each file includes:
- the slides owned by that member
- the folders and files they should revise deeply
- the exact technical points they must know
- full explanations in simple viva-friendly language
- likely teacher questions and strong answers
- quick revision facts for the last 5 minutes before presentation

Recommended use:
1. Each member reads only their own file first.
2. Then all members revise the common project flow and basic facts together.
3. Practice speaker handoff so the presentation feels continuous.
